# 微短剧叙事技能

## 功能描述

该技能能够将文学内容转化为适合短视频平台的微短剧叙事结构，包括：

1. **强节奏掌控**：设计紧凑的剧情发展，避免拖沓
2. **钩子设计**：在每集开场30秒内设置强钩子，吸引观众继续观看
3. **碎片化叙事**：适应短视频平台用户的观看习惯，便于碎片化消费
4. **每集独立成篇**：确保单集内容完整，同时保持整体剧情连贯
5. **悬念设置**：在每集结尾设计悬念，引导观众观看下一集

## 使用场景

- 当需要将文学内容改编为微短剧时
- 当需要设计微短剧的叙事结构时
- 当需要优化微短剧的节奏和悬念时
- 当需要适应短视频平台的叙事要求时

## 调用示例

```
调用 short-drama-narration 技能，将以下文学内容改编为12集微短剧的叙事大纲：

[文学内容摘要]

请输出：
1. 整体叙事结构设计
2. 每集核心冲突和钩子
3. 每集结尾悬念
4. 节奏控制建议
```

## 技术实现

- 基于短视频平台的叙事规律，设计适合的叙事结构
- 运用节奏理论，控制剧情发展的快慢
- 结合观众心理学，设计有效的钩子和悬念
- 考虑碎片化观看习惯，确保每集独立成篇

## 注意事项

- 适应不同平台的观众偏好（如抖音、快手、B站等）
- 根据IP类型调整叙事风格（如甜宠、悬疑、都市等）
- 保持原IP的核心精神，避免过度改编
- 考虑拍摄可行性，避免过于复杂的叙事结构

该技能为微短剧改编提供了关键的叙事框架，确保改编后的内容符合短视频平台的要求，能够吸引观众并保持高收视率。
